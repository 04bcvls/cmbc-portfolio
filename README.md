# cmbc Portfolio

A modern, interactive developer portfolio website built with React, Tailwind CSS, and Vite.

## 🚀 Features

- **Modern Design** - Glassmorphism effects with liquid glass aesthetic
- **Responsive Layout** - Fully responsive design for mobile, tablet, and desktop
- **Smooth Animations** - Scroll-triggered animations and interactive elements
- **Dark Theme** - Professional dark color scheme with purple accents
- **Performance Optimized** - Built with Vite for fast development and production builds
- **Back-to-Top Button** - Smooth scroll animation with fade-in/out effect
- **Auto-Advancing Carousel** - Portrait carousel with manual and automatic navigation
- **Tech Stack Display** - Showcase of skills with devicon logos

## 🛠️ Tech Stack

- **Frontend Framework** - React 18.2.0
- **Build Tool** - Vite 5.1.0
- **Styling** - Tailwind CSS 3.4.1
- **Fonts** - Google Fonts (Syne, DM Sans, Montserrat)
- **Animations** - CSS Keyframes
- **Icons** - SVG & Devicon CDN

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/04bcvls/CivilesJuniorPortfolio.git

# Navigate to the project
cd CivilesJuniorPortfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```
src/
├── components/
│   ├── About.jsx          # About section with carousel
│   ├── Affiliation.jsx    # Organization affiliations
│   ├── BackToTop.jsx      # Scroll-to-top button
│   ├── Hero.jsx           # Landing hero section
│   ├── Navbar.jsx         # Navigation bar
│   ├── Projects.jsx       # Project portfolio
│   └── TechStacks.jsx     # Skills and technologies

│   └── useInView.js       # Scroll animation hook
├── App.jsx                # Main app component
├── index.css              # Global styles
└── main.jsx               # Entry point
```
